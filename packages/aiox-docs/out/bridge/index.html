Princípios de Design

🛡️

Fail-Open Architecture

Todos os sistemas de memória degradam gracefully. Timeout ou erro → retorna array vazio. Nunca bloqueia execução.

🎚️

Bracket-Aware Injection

Contexto restante determina profundidade da memória. 60-100% = skip. 0-25% = full content + handoff warning.

🔒

Hash-Based Sync

SHA-256 comparison garante que apenas arquivos new/modified são processados. Zero overhead para unchanged files.

🎯

Sector-Filtered Recall

Cada agente recebe apenas memórias relevantes ao seu papel. @dev → procedural/semantic. @qa → reflective/episodic.

⚡

15ms Timeout Budget

Synapse engine tem budget máximo de 15ms para memory retrieval. Session-level cache por {agentId}-{bracket}.

🔄

Pattern Promotion

Padrões vistos em 3+ agents são promovidos de MEMORY.md individual para CLAUDE.md ou .claude/rules/ (global).
