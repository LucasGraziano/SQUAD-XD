Commands

Comandos & Ativacao

3 formas de interagir: @agent para ativar agentes, *comando para executar tasks, e /slash para ferramentas rapidas.

Como Ativar Agentes

@dev

Sintaxe: @agent-name

Ativa agente AIOX core

/AIOX:agents:architect

Sintaxe: /AIOX:agents:agent-name

Ativa via slash command

/atlas

Sintaxe: /atlas

Ativa Commander Atlas (Low-Ticket Squad)

Comandos de Agente

Prefixo * — executam tasks estruturadas do agente ativo

*help

Mostrar todos os comandos disponíveis do agente ativo

*create-story

Criar nova user story a partir de PRD/epic

*task {name}

Executar task específica do agente

*workflow {name}

Iniciar workflow multi-agente

*plan

Criar plano de implementação

*exit

Sair do modo agente ativo

*kb

Ativar Knowledge Base mode (AIOX Method)

*status

Ver contexto e progresso atual

*guide

Guia completo do agente ativo

Auto Skills

Skills semânticas — ativam automaticamente por contexto. Sem prefixo. A IA decide quando usar com base na descrição.

design-aware-uidesign

Carrega DESIGN.md do squad antes de gerar qualquer UI. Previne AI slop (bento grids, neon, glassmorphism). Aplica tokens de cor, tipografia e espaçamento corretos.

criar componentebuild UIlanding pagedashboardformbuttoncardstylefrontendlayout
brand-recommenderdesign

Recomenda o melhor brand da biblioteca de 70 DESIGN.md files com base no mood, propósito ou público do projeto.

inspiração de designqual brand usarreferência visualcomo o Linearcomo o Stripeestilo similar
knowledge-routerknowledge

Roteia para o expert DNA, dossier ou playbook correto da knowledge layer. Evita respostas genéricas em copy, oferta, tráfego, design, produto e SaaS.

copyofertafuniltráfegovendasestratégiaposicionamentodesign systemprodutosaas
atlas-briefingproductivity

Gera status operacional rápido do sistema: squads ativos, último commit, próximas ações. Ativa ao iniciar sessão.

statuso que está acontecendocontextoonde estamosbriefingretomaroverview
context-hygieneproductivity

Recomenda /clear, /compact ou /checkpoint nos momentos certos para manter tokens baixos e contexto limpo.

Slash Commands

Prefixo / — ferramentas de produtividade e geracao de conteudo

/atlas
productivity

Ativa o agente Atlas, Chief of Staff do Low-Ticket Squad. Coordena todos os 28 agentes especializados.

Uso
/atlas/atlas deploy-squad/atlas status
/atlas-briefing
productivity

Gera briefing operacional diario com integração Notion. Mostra tasks, deadlines, blockers e foco recomendado.

Uso
/atlas-briefing/atlas-briefing --compact
/custos
productivity

Dashboard consolidado de todos os serviços de IA com custos estimados e limites gratuitos.

Uso
/custos
/weekly-prep
productivity

Preparação automática da pauta da weekly. Consolida progresso, blockers e próximos passos.

Uso
/weekly-prep
/review-day
productivity

Fechamento de sessão + log de produtividade. Registra o que foi feito e planeja próxima sessão.

Uso
/review-day
/ebook
content

Gerador de ebooks/PDFs profissionais com identidade visual. Puppeteer + HTML/CSS templates.

Uso
/ebook modulo-1/ebook modulo-2/ebook tracker/ebook custom "Titulo"
/research
content

Pesquisa científica com citações reais. Busca artigos, livros, meta-análises e compila resumo estruturado.

Uso
/research "diastasis recti exercise"/research --deep "pelvic floor postpartum"
/video-tools
tools

Ferramentas de processamento de vídeo via FFmpeg. Concat, audio, trim, export para Meta Ads, thumbnails.

Uso
/video-tools concat/video-tools add-audio/video-tools trim/video-tools export/video-tools thumbnail
/checkpoint
productivity

Cria snapshot do contexto da conversa atual. Salva o que fizemos, decisões, próximos passos — tudo em .claude/checkpoints/ para retomar depois.

Uso
/checkpoint/checkpoint "Titulo custom"/checkpoint --tag feature,ui
/recall
productivity

Retoma contexto de sessões anteriores. Lista, busca e carrega checkpoints salvos — permite continuar qualquer conversa em outro ambiente.

Uso
/recall/recall --last/recall --search "landing page"/recall 3
/ingest
knowledge

Extrai DNA cognitivo de experts a partir de vídeos, transcrições ou texto. Comprime em 5 camadas (Filosofias, Modelos Mentais, Heurísticas, Frameworks, Metodologias) com max ~700 tokens.

Uso
/ingest/ingest --mode simple/ingest --merge
/dossier
knowledge

Gera dossiers cross-source com convergências e divergências entre múltiplos experts. ~1000 tokens cada.

Uso
/dossier {tema}/dossier --auto {domain}/dossier --list
/playbook
knowledge

Consulta ou gera playbooks operacionais a partir de dossiers. Frameworks acionáveis com ~800 tokens cada.

Uso
/playbook {nome}/playbook --generate {tema} --from {dossier}/playbook --list
/conclave
knowledge

Deliberação multi-agente v2 com CRITIC (scoring 0-100), DEVIL'S ADVOCATE (stress test) e SYNTHESIZER (GO/NO-GO/CONDITIONAL).

Uso
/conclave {pergunta}/conclave --v1 {pergunta}
/wiki
knowledge

Sistema de conhecimento compounding (Karpathy-style). Ingere fontes no wiki, consulta conhecimento acumulado, faz lint de qualidade. RAW/ (imutável) + WIKI/ (artigos compilados).

Uso
/wiki-ingest <url|path>/wiki-query "<pergunta>"/wiki-lint/wiki-status
