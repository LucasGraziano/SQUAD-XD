## Descrição

Implementar o motor de classificação baseado em regras locais (Camada 1). Extrair padrões das 263 transações existentes para criar regras automáticas e implementar o feedback loop para aprendizado contínuo.

## Acceptance Criteria

- [ ] ~50 regras de classificação extraídas do histórico
- [ ] Regras armazenadas na aba "Regras" com padrão, tipo, categoria, hits
- [ ] Função `classifyByRules(description)` retorna categoria + confiança
- [ ] Match types suportados: exact, contains, starts_with
- [ ] onEdit trigger: quando usuário corrige categoria → nova regra criada
- [ ] Regras testadas contra as 263 transações existentes com ≥70% match

## Tasks

- [ ] Criar aba "Regras" no Google Sheet (padrão, match_type, tipo, categoria, hits, origem)
- [ ] Analisar 263 transações e extrair padrões desc→categoria
- [ ] Popular ~50 regras seed (origin: seed)
- [ ] Implementar `classifyByRules()` em Apps Script
- [ ] Implementar match por prioridade (exact > contains > starts_with)
- [ ] Implementar `onEdit` trigger para feedback loop
- [ ] Implementar `learnFromCorrection()` — cria/atualiza regra quando usuário edita
- [ ] Testar: rodar classificação nas 263 transações e medir hit rate
- [ ] Ajustar regras até atingir ≥70% coverage
- [ ] Documentar regras na aba Config

## Definition of Done

- Motor de regras funcional com ≥70% de cobertura nas transações existentes
- Feedback loop testado (editar categoria → regra criada)
- Hit counter incrementando corretamente
- Prioridade de match funcionando (exact primeiro)
