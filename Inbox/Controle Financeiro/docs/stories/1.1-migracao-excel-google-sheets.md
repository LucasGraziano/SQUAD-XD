## Descrição

Criar o Google Sheet estruturado e migrar os 263 registros do Excel atual, garantindo zero perda de dados e estrutura otimizada para automação futura.

## Acceptance Criteria

- [ ] Google Sheet criado com 3 abas: Transações, Dashboard, Config
- [ ] 263 transações migradas do Excel com todos os campos
- [ ] Colunas redundantes eliminadas (Ano, Mês, Mês_extenso → auto-calculadas)
- [ ] Colunas novas adicionadas: ID, Status, Confiança, Hash, Fonte
- [ ] Aba Config populada com 13 categorias + cores + orçamentos default
- [ ] Formatação condicional configurada (verde/amarelo/vermelho por status)
- [ ] Data validation em Tipo e Categoria (dropdowns)
- [ ] Validação: contagem = 263, soma dos valores = R$ 85.269,53
