## Descrição

Criar dashboard na aba "Dashboard" do Google Sheets com KPIs, gráficos de categorias, tendências e budget tracking. Deve atualizar automaticamente quando novas transações são adicionadas.

## Acceptance Criteria

- [x] KPIs no topo: Receita, Despesa, Aportes, Saldo do mês
- [x] Comparativo com mês anterior (▲/▼ %)
- [x] Gráfico donut: gastos por categoria (mês atual)
- [x] Gráfico barras: tendência mensal (últimos 6 meses)
- [x] Budget vs Actual: barras de progresso por categoria
- [x] Filtro por mês/ano (dropdown)
- [x] Taxa de poupança calculada (aportes / receita)
- [x] Atualização automática (fórmulas, sem manual refresh)

- Dashboard visual e funcional com todos os gráficos
