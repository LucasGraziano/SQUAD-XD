## Descrição

Implementar a Camada 2 de classificação usando Ollama (Llama 3.2) rodando localmente. Como o Ollama roda em localhost e Apps Script não acessa localhost, a abordagem será um **script Python** que busca transações pendentes do Sheets, classifica via Ollama, e atualiza de volta.

## Premissas

- **Custo: R$ 0** — Ollama é local e gratuito
- **Ollama já instalado:** `C:\Users\Graziano\AppData\Local\Programs\Ollama\ollama.exe`
- **Modelo:** Llama 3.2 (já pulled)
- **Endpoint:** `http://localhost:11434/api/generate`

## Arquitetura

```
Google Sheets (Transações com 🔴 status)
    │
    │  (1) Script Python lê via Google Sheets API
    ▼
Script Python local
    │
    │  (2) Para cada 🔴: chama Ollama localhost:11434
    ▼
Ollama (Llama 3.2)
    │
    │  (3) Retorna: categoria + confiança
    ▼
Script Python
    │
    │  (4) Atualiza Sheets: 🔴 → 🟡 (sugerido) ou 🟢 (alta confiança)
    ▼
Google Sheets (atualizado)
```

## Acceptance Criteria
