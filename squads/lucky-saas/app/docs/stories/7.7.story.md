## User Story

Como corretor, quero ver o histórico completo de alterações de uma proposta — quais valores foram mudados, quando, e qual versão foi enviada ao cliente — para ter rastreabilidade e poder explicar ao cliente por que o preço mudou.

---

## Value Proposition

- PRD FR-04: "Histórico de versões da proposta" — especificado, não implementado
- Corretor frequentemente negocia: prêmio cai, cobertura muda, seguradora troca
- Sem histórico, o corretor não sabe o que foi enviado vs. o que está aberto
- Diferencial vs. planilha: Excel não tem versioning automático
- Rastreabilidade para resolução de disputas ("mas eu enviei R$1.200, não R$1.400")

---

## Acceptance Criteria

- [ ] **AC-1:** Cada alteração em `quote_items` (prêmio, cobertura, seguradora, `is_recommended`) é registrada em `quote_item_history`
- [ ] **AC-2:** Painel "Histórico" disponível na página de detalhes de uma cotação (aba ou accordion)
- [ ] **AC-3:** Histórico exibe: data/hora, o que mudou (campo), valor anterior → valor novo, quem fez (sempre o broker)
- [ ] **AC-4:** Snapshot automático ao mudar status para `sent` — registro imutável do que foi enviado ao cliente
- [ ] **AC-5:** Interface de diff visual: campo alterado destacado em amarelo (valor antes) → verde (valor depois)
- [ ] **AC-6:** Máximo de 50 entradas de histórico por cotação (FIFO — mais antigas descartadas)
- [ ] **AC-7:** Histórico read-only — não editável pelo corretor

## Technical Spec

### DB — Migration

```sql
-- supabase/migrations/20260514000005_quote_history.sql
CREATE TABLE quote_item_history (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  quote_id UUID NOT NULL REFERENCES quote_requests(id) ON DELETE CASCADE,
  quote_item_id UUID REFERENCES quote_items(id) ON DELETE SET NULL,
  broker_id UUID NOT NULL REFERENCES brokers(id) ON DELETE CASCADE,
  field_name TEXT NOT NULL,
  old_value TEXT,
  new_value TEXT,
  changed_at TIMESTAMPTZ DEFAULT now(),
  snapshot_type TEXT -- NULL = edição normal, 'sent' = snapshot ao enviar
);
ALTER TABLE quote_item_history ENABLE ROW LEVEL SECURITY;
CREATE POLICY "broker own" ON quote_item_history USING (broker_id = auth.uid());
```
