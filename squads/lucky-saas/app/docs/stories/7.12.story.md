## User Story

Como corretor, quero ser alertado automaticamente quando detectar oportunidades de cross-sell na minha carteira — por exemplo, "3 clientes com seguro de vida mas sem seguro auto" — para contatar esses clientes com uma sugestão relevante antes que eles procurem outro corretor.

---

## Value Proposition

- Corretor tem o dado mas não tem a inteligência: sabe quem é cliente, mas não vê oportunidades
- Cross-sell com cliente existente custa 5-7× menos que adquirir novo cliente (Harvard Business Review)
- Hoje o corretor precisa cruzar manualmente cliente × apólices para achar oportunidades — não faz
- Automatizar essa detecção = mais comissão sem mais esforço do corretor
- Referência: Segfy não tem isso; Quiver IN não tem isso — diferencial claro
- Segmentos de cross-sell: Vida → Auto, Auto → Vida, qualquer ramo → Residencial

---

## Acceptance Criteria

- [ ] **AC-1:** Seção "Oportunidades de Cross-sell" no dashboard, exibindo até 5 oportunidades priorizadas por número de clientes
- [ ] **AC-2:** Regras de cross-sell detectadas automaticamente:
  - Clientes com apólice de Vida ativa mas sem apólice de Auto ativa
  - Clientes com apólice de Auto ativa mas sem apólice de Vida ativa
  - Clientes com qualquer apólice ativa mas sem apólice Residencial ativa
  - Clientes com apólice de Saúde ativa mas sem apólice de Vida ativa
- [ ] **AC-3:** Cada oportunidade exibe: ícone do ramo sugerido, quantidade de clientes afetados, exemplo: "4 clientes com Auto mas sem Vida"
- [ ] **AC-4:** Botão "Ver clientes" em cada oportunidade → abre lista filtrada dos clientes afetados
- [ ] **AC-5:** Botão "Contatar todos" → gera mensagem WhatsApp pré-formatada para copiar: "Olá, tenho uma proposta de [ramo] que pode te interessar..."
- [ ] **AC-6:** Oportunidades são calculadas em tempo real (a cada load da dashboard, sem tabela separada)
- [ ] **AC-7:** Se não há oportunidades detectadas, a seção não é exibida (sem empty state para cross-sell)
- [ ] **AC-8:** Disponível apenas para plano Profissional+ (`<PlanGate requiredPlan="profissional">`)

### Exemplo de Query

```sql
-- Clientes com Vida mas sem Auto
SELECT c.id, c.nome
FROM clients c
JOIN apolices a ON a.client_id = c.id AND a.ramo = 'vida' AND a.vigencia_fim > NOW()
WHERE c.broker_id = $brokerId
  AND NOT EXISTS (
    SELECT 1 FROM apolices a2
    WHERE a2.client_id = c.id AND a2.ramo = 'auto' AND a2.vigencia_fim > NOW()
  )
```
