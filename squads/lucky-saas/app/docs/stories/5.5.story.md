## User Story

Como corretor, quero ver relatórios detalhados da minha carteira e um forecast de comissões dos próximos 3 meses, para planejar meu caixa e identificar onde estou crescendo ou perdendo.

---

## Value Proposition

- Corretor hoje não sabe quanto vai receber no próximo mês — vive no improviso
- Forecast de comissões = planejamento financeiro pessoal e profissional
- Relatório de retenção por ramo = decisão de onde prospectar mais
- Diferencial claro vs. Facilita Corretor e Maiseg (ambos têm apenas relatórios básicos)
- Feature exclusiva do plano Profissional (R$97) — driver de upgrade

## Technical Spec

### Data Sources

Todas as queries são sobre a tabela `policies` existente:

```sql
-- Forecast: apólices ativas que vencem nos próximos 90d × commission_pct
SELECT
  DATE_TRUNC('month', end_date) AS month,
  SUM(COALESCE(commission_expected, premium_total * commission_pct / 100)) AS forecast_commission
FROM policies
WHERE broker_id = $broker_id
  AND status = 'active'
  AND end_date BETWEEN NOW() AND NOW() + INTERVAL '90 days'
GROUP BY 1 ORDER BY 1;

-- Retenção por ramo
SELECT
  insurance_type,
  COUNT(*) FILTER (WHERE status = 'active') AS renewed,
  COUNT(*) FILTER (WHERE status = 'expired') AS expired,
  ROUND(COUNT(*) FILTER (WHERE status = 'active')::numeric /
    NULLIF(COUNT(*), 0) * 100, 1) AS retention_rate
FROM policies
WHERE broker_id = $broker_id
GROUP BY 1;
```
