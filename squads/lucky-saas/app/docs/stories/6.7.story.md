## User Story

Como corretor satisfeito com o Premia, quero indicar colegas e ganhar 1 mês grátis por cada indicado que assinar, para ter um incentivo concreto de divulgar o produto na minha rede.

---

## Value Proposition

- Canal de menor CAC: referral orgânico via pares (corretores indicam corretores)
- GTM playbook define referral como canal principal da Fase 1 (meses 1–3)
- Mecânica: indicador ganha 1 mês grátis; indicado ganha trial de 21 dias (vs. 14 padrão)
- Sem referral program, crescimento depende 100% de ads pagos desde o dia 1

---

## Acceptance Criteria

- [ ] **AC-1:** Cada corretor tem um código de referral único (`/ref/{code}`) gerado na criação da conta
- [ ] **AC-2:** Link de indicação disponível em `/dashboard` em seção "Indique e ganhe" — botão "Copiar link" + botão "Enviar pelo WhatsApp" (link pré-formatado: "Uso o Premia para gerenciar minha carteira de seguros. Experimente grátis por 21 dias: {link}")
- [ ] **AC-3:** Visitante que acessa `/ref/{code}` tem o código salvo em cookie (30 dias) e é redirecionado para landing page com banner: "Você foi indicado por um corretor — seu trial é de 21 dias"
- [ ] **AC-4:** No signup, se cookie de referral presente, registrar `referral_code_id` na conta do novo corretor
- [ ] **AC-5:** Quando indicado converte trial → pago (evento `checkout.session.completed` do Stripe), registrar em `referral_conversions` com `referred_broker_id` e `converted_at`
- [ ] **AC-6:** Após 30 dias de pagamento confirmado do indicado, aplicar crédito de 1 mês no plano do indicador (via Stripe `customer.balance` ou coupon de 1 mês)
- [ ] **AC-7:** Dashboard do indicador mostra: total de indicações, total convertidas, créditos ganhos
- [ ] **AC-8:** E-mail automático ao indicador quando indicado converte: "🎉 [Nome] assinou o Premia! Seu crédito de 1 mês foi aplicado na próxima fatura."

## Technical Spec

### Database

```sql
CREATE TABLE referral_codes (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  broker_id UUID NOT NULL UNIQUE REFERENCES brokers(id),
  code TEXT NOT NULL UNIQUE,
  created_at TIMESTAMPTZ DEFAULT now()
);

CREATE TABLE referral_conversions (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  referral_code_id UUID NOT NULL REFERENCES referral_codes(id),
  referred_broker_id UUID NOT NULL UNIQUE REFERENCES brokers(id),
  converted_at TIMESTAMPTZ,
  reward_applied_at TIMESTAMPTZ,
  created_at TIMESTAMPTZ DEFAULT now()
);

ALTER TABLE brokers ADD COLUMN referral_code_id UUID REFERENCES referral_codes(id);
```
