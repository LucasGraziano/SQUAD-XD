## User Story

Como corretor, quero ver no perfil de cada cliente o histórico completo de todas as apólices que já tiveram — incluindo apólices encerradas, substituídas por renovação e a cadeia de renovações — para ter rastreabilidade total e poder explicar ao cliente a evolução da sua cobertura ao longo do tempo.

---

## Value Proposition

- Corretor de longa data tem clientes com 5-10+ apólices renovadas — perdidos em planilhas
- "Qual era o prêmio do ano passado?" é uma pergunta que o corretor não consegue responder rapidamente
- Histórico de renovações = confiança do cliente + rastreabilidade em disputas
- Diferencial vs planilha: Excel não tem o link de renovação entre apólices
- Sem nova tabela: usa `renewed_by_apolice_id` adicionado na Story 7.5 para construir a cadeia

---

## Acceptance Criteria

- [ ] **AC-1:** Aba "Histórico de Apólices" no perfil do cliente (`/clientes/[id]`)
- [ ] **AC-2:** Listagem de TODAS as apólices do cliente (ativas e encerradas), ordenadas por `vigencia_fim DESC`
- [ ] **AC-3:** Para cada apólice no histórico: número, seguradora, ramo, prêmio, vigência (início-fim), status atual
- [ ] **AC-4:** Apólices que fazem parte de uma cadeia de renovação são agrupadas visualmente com seta "Renovada por → [Apólice #X]"
- [ ] **AC-5:** Badge de status por apólice: "Ativa" (verde), "Expirada" (cinza), "Cancelada" (vermelho), "Renovada" (azul com link para nova)
- [ ] **AC-6:** Link "Ver detalhes" em cada apólice → navega para `/apolices/[id]`
- [ ] **AC-7:** Totalizador no topo: "X apólices ao total — Y ativas — prêmio total atual: R$ Z"
- [ ] **AC-8:** Histórico inclui apólices de TODOS os ramos do cliente (não apenas o ramo atual)

```typescript
// src/lib/clients/renewal-history.ts
export interface ApoliceHistoryItem {
  id: string
  numero_apolice: string
  seguradora: string
  ramo: string
  premio: number
  vigencia_inicio: string
  vigencia_fim: string
  status: 'ativa' | 'expirada' | 'cancelada' | 'renovada'
  renewed_by_apolice_id: string | null
  renewal_quote_id: string | null
}

export async function getClientApoliceHistory(
  clientId: string,
  brokerId: string
): Promise<ApoliceHistoryItem[]>

// Query: SELECT * FROM apolices
// WHERE client_id = $clientId AND broker_id = $brokerId
// ORDER BY vigencia_fim DESC
// Status derivado:
//   vigencia_fim > today → 'ativa'
//   renewed_by_apolice_id IS NOT NULL → 'renovada'
//   status = 'cancelled' → 'cancelada'
//   else → 'expirada'
```
