## User Story

Como corretor, quero ver um extrato cronológico de todas as comissões — esperadas e recebidas — filtrado por seguradora, ramo e período, para ter controle financeiro completo do meu negócio.

---

## Value Proposition

- BrokerOne tem "Extrato de Participações" e "Participações Corretores" como relatórios separados
- Premia tem financeiro mas sem histórico granular de comissão por apólice
- Feature de controle financeiro que retém corretores a longo prazo

## Technical Spec

### Database

```sql
-- Adicionar campos de comissão recebida nas apólices
ALTER TABLE policies
  ADD COLUMN IF NOT EXISTS commission_received NUMERIC(12,2),
  ADD COLUMN IF NOT EXISTS commission_received_at DATE,
  ADD COLUMN IF NOT EXISTS commission_status TEXT DEFAULT 'prevista'
    CHECK (commission_status IN ('prevista', 'recebida', 'vencida'));
```

### Server Actions

- `getCommissionStatement({ from, to, seguradora?, ramo?, status? })` → lista paginada
- `markCommissionReceived(policyId, { amount, date })` → update
- `getCommissionSummary(params)` → totalizadores

### UI Components

- `ExtratoTab` — nova aba em `/financeiro`
- `CommissionRow` — linha da tabela com botão "Marcar Recebida"
- `MarkReceivedModal` — modal para informar valor e data reais
- `CommissionSummaryBar` — barra de totalizadores no topo
- Gráfico esperado vs recebido (reutiliza padrão existente)
