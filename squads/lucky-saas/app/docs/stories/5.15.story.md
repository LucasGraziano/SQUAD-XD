## User Story

Como corretor migrando para o Premia, quero importar minha carteira inteira via planilha CSV/Excel, para não precisar cadastrar apólice por apólice manualmente e começar a usar o sistema imediatamente.

---

## Value Proposition

- Blocker de adoção para corretoras com 50+ apólices
- BrokerOne tem "Importação de Arquivos" como feature ativa (14 pendentes no dashboard observado)
- Remove principal objeção de migração: "mas vou perder minha carteira toda"

---
