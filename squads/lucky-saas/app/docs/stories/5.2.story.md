## User Story
console.cloud.google.com
Como corretor, quero gerar uma proposta comercial em PDF com os dados da apólice/cotação, com minha marca, para enviar ao cliente de forma profissional — sem precisar montar no Word.

---

## Value Proposition

- Corretores independentes perdem negócios por falta de materiais profissionais
- Proposta PDF com marca do corretor = diferencial de percepção de valor
- Substitui horas de formatação manual no Word/Canva
- Feature exclusiva do plano Profissional (R$97) — driver de upgrade

---

## Acceptance Criteria

- [ ] **AC-1:** Botão "Gerar Proposta PDF" disponível na página de detalhe de uma apólice
- [ ] **AC-2:** PDF inclui: logo/nome da corretora, dados do cliente, dados da apólice (ramo, seguradora, vigência, prêmio, coberturas principais), validade da proposta (configurable: 7/15/30 dias)
- [ ] **AC-3:** Template visual profissional — cores neutras, tipografia limpa, sem visual genérico
- [ ] **AC-4:** PDF gerado no servidor (não no browser) via biblioteca React PDF ou Puppeteer
- [ ] **AC-5:** Download direto do arquivo (`proposta-[nome-cliente]-[data].pdf`)
- [ ] **AC-6:** Opção de enviar link de download por e-mail diretamente da interface
- [ ] **AC-7:** Configurações da corretora (logo, nome, SUSEP, telefone, email) usadas no header do PDF
- [ ] **AC-8:** PDF legível em mobile (layout responsivo adaptado para A4)
- [ ] **AC-9:** Funcionalidade bloqueada no plano Solo — exibe upgrade wall
- [ ] **AC-10:** Geração leva menos de 5 segundos para documento padrão

---

## Technical Spec

### Stack

- `@react-pdf/renderer` — geração de PDF server-side via React
- Server Action: `generateProposalPDF(policyId)` → retorna blob
- Route handler: `GET /api/pdf/proposal/[policyId]` para download
- Configurações do corretor: tabela `broker_settings` (logo_url, susep, phone, etc.)

### Template

```
[HEADER]
  Logo corretora | Nome da Corretora | SUSEP: XXXXXXXX
  
[CLIENTE]
  Nome: João Silva
  CPF: XXX.XXX.XXX-XX
  Email / Telefone

[PROPOSTA]
  Produto: Seguro Auto — Veículo: Toyota Corolla 2022
  Seguradora: Porto Seguro
  Vigência: 01/06/2026 a 01/06/2027
  Prêmio: R$ 2.847,00/ano
  
[COBERTURAS]
  ✓ Colisão / Incêndio / Roubo
  ✓ RCFV — Danos a terceiros
  ✓ APP — Morte e Invalidez
  
[VALIDADE]
  Esta proposta é válida até: 16/05/2026
  
[FOOTER]
  Corretor: Lucas Graziano | lucas@premia.app | (11) 99999-9999
  Gerado por Premia — premia.app
```
