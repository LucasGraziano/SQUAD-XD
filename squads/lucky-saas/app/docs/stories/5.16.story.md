## User Story

Como corretor, quero ver métricas de performance do meu negócio — taxa de emissão, tempo médio de fechamento, e renovações ganhas vs perdidas — para identificar onde estou perdendo negócios e melhorar minha operação.

---

## Value Proposition

- BrokerOne tem "Desempenho Técnico/Operacional" como relatório dedicado
- Corretor hoje sabe quanto ganhou mas não sabe *onde perdeu*
- Feature de alto valor percebido: "sou um profissional que monitora performance"

## Technical Spec

### Lógica de cálculo (sem nova tabela)

```typescript
// Taxa de emissão: requer dados de proposals (Story 5.13)
// Fallback sem proposals: apólices criadas / leads totais no período

// Tempo médio: policies.created_at vs leads.created_at (para leads convertidos)

// Taxa de renovação:
// apólices vencidas no período = policies WHERE end_date BETWEEN start AND end
// renovadas = policies criadas após outra do mesmo cliente/ramo

// Ticket médio: AVG(premium_total) WHERE status = 'ativa' AND created_at IN period
```

### Server Actions

- `getOperationalMetrics({ period: '3m'|'6m'|'12m' })` → objeto com todos os KPIs
- Retorna dados mensais para gráficos + totais do período

### UI Components

- `DesempenhoTab` — nova aba em RelatoriosClient
- `KPICard` — card com valor atual + variação vs período anterior
- `MetricChart` — gráfico de barras mensal (reutiliza padrão do relatorio existente)
