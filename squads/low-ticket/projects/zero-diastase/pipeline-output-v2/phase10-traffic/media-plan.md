## Estratégia de Mídia — 3 Fases

### Fase 1 — Blast & Read (Semana 1-2)

**Objetivo:** Testar criativos, validar CPL e CPA, sair da learning phase.

| Aspecto | Detalhe |
|---------|---------|
| **Budget** | $60 (~R$330) — 47% do seed |
| **Budget diário** | ~$4.30/dia |
| **Estrutura** | 1 CBO, 1 adset broad LATAM, 3-4 ads |
| **Objetivo campanha** | Conversions → Lead (QuizComplete) |
| **Audiência** | Broad — Mulheres 25-45, all LATAM hispano |
| **Placements** | Auto (Feed + Stories + Reels) |
| **Criativos** | 3-4 variações (ver seção Criativos) |
| **Meta** | CPL < $2.00, identificar país/criativo winner |

**Por que Lead primeiro:** Com $4/dia, otimizar para Purchase não sai da learning phase (precisa ~50 conversões/semana). Otimizar para Lead (quiz complete) gera volume suficiente para aprendizado.

**Decisão Semana 2:** Se CPL < $1.50 e taxa quiz→venda > 2%, migrar para Purchase. Senão, manter Lead e otimizar copy/funnel.

### Fase 2 — Double Down (Semana 3-4)

**Objetivo:** Escalar o que funciona, matar o que não funciona.

| Aspecto | Detalhe |
|---------|---------|
| **Budget** | $67 (~R$370) — 53% do seed + primeiros ROIs |
| **Budget diário** | ~$4.80/dia + ROI reinvestido |
| **Ação** | Kill criativos com CPA > $12, duplicar budget no winner |
| **Otimização** | Testar novo criativo baseado nos dados de Fase 1 |
| **Audiência** | Manter broad, mas analisar breakdown por país |

### Fase 3 — Scale or Kill (Mês 2+)

**Objetivo:** Escalar com ROI reinvestido ou pivotar.

| Cenário | CPA | Ação |
|---------|-----|------|
| **Scale** | ≤ $6 | Aumentar budget 20%/semana, testar novos criativos |
| **Optimize** | $6-$10 | Otimizar funnel (quiz, TSL, copy), manter budget |
| **Pivot** | $10-$12 | Testar novos ângulos, mudar hooks, revisar oferta |
| **Kill** | > $12 consistente | Pausar, re-avaliar posicionamento ou produto |

## Targeting Detalhado

### Audiência Principal (D1)

| Parâmetro | Valor |
|-----------|-------|
| **Idade** | 25-45 |
| **Gênero** | Mulheres |
| **Geo** | Argentina, Colombia, México, Chile, Peru, Ecuador, Guatemala, República Dominicana, Costa Rica, Panamá, Uruguay, Paraguay, Bolivia, Honduras, El Salvador, Nicaragua, Puerto Rico |
| **Idioma** | Espanhol (qualquer) |
| **Interesses** | NENHUM — 100% broad |
| **Exclusões** | Nenhuma (broad máximo) |
| **Placement** | Automatic (recomendado para budget baixo) |

### Por que Broad?

1. **Budget baixo** ($4/dia) → segmentação detalhada fragmenta demais
2. **Meta algoritmo** otimiza melhor com audiência grande + pixel data
3. **LATAM** tem CPMs baixos ($3-6) → broad ainda é eficiente
4. **Quiz funciona como filtro** — quem não é o público abandona no quiz, não no ad

### Análise de Países por CPM (referência)

| País | CPM Est. | Potencial | Nota |
|------|----------|-----------|------|
| Colômbia | $3.00-4.00 | Alto | Maior volume, CPM baixo |
| México | $3.50-5.00 | Alto | Maior população, mais competição |
| Argentina | $2.00-3.50 | Médio-Alto | CPM muito baixo, instabilidade cambial |
| Peru | $2.50-4.00 | Médio | Bom CPM, mercado receptivo |
| Chile | $5.00-8.00 | Médio | CPM mais alto, mas maior poder de compra |
| Ecuador | $3.00-4.50 | Médio | Dolarizado (USD nativo) |
| Guatemala | $2.00-3.00 | Baixo-Médio | CPM muito baixo, menor poder de compra |
| Rep. Dominicana | $3.50-5.00 | Médio | Caribe, boa penetração mobile |
| Puerto Rico | $8.00-12.00 | Baixo | CPM alto (USA pricing) |

**Estratégia:** Deixar Meta encontrar os CPMs mais eficientes. Analisar breakdown semanal para entender onde as vendas acontecem.
