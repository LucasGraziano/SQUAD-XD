## 7. REGRAS DE ESCALA

### Escala Vertical (mais budget no winner)
```
Winner identificado (3+ conversões, CPA < $10)
  → Dia 1: Duplicar adset, dobrar budget ($5 → $10)
  → Dia 3: Se manteve CPA → +30% ($10 → $13)
  → Dia 5: Se manteve → +30% ($13 → $17)
  → Dia 7: Se manteve → +30% ($17 → $22)

  Se CPA subir > 20% → PARAR escala, manter budget atual
  NUNCA aumentar mais que 30% de uma vez
```

### Escala Horizontal (novos públicos)
```
Winner validado em Colômbia
  → Duplicar campanha inteira para México (mesmos ads)
  → Testar por 5 dias com US$15/dia
  → Se funcionar → adicionar Porto Rico

  Cada país novo = novo teste (não assumir que funciona)
```

### Criativo Fatigue (quando parar de funcionar)
```
Sinais: CTR caindo 2 dias seguidos, CPA subindo
  → NÃO editar o ad (quebra otimização)
  → Duplicar adset + trocar criativos por novos
  → Cabral gera 2-3 hooks novos, Lucas produz criativos
  → Rotacionar a cada 7-14 dias
```
