## 1. PALETA DE CORES

### Cores Primárias

| Nome | Hex | Uso |
|------|-----|-----|
| **Blush** | `#D4A5A5` | Cor principal — headlines, destaques, botões secundários |
| **Nude Warm** | `#F5E6D3` | Backgrounds suaves, cards, seções alternadas |
| **White Warm** | `#FFFAF5` | Background principal — nunca branco puro |
| **Gold Soft** | `#C9A96E` | Acentos — ícones, bordas, badges, preço |

### Cores de Suporte

| Nome | Hex | Uso |
|------|-----|-----|
| **Text Dark** | `#2D2D2D` | Texto principal — nunca preto puro (#000) |
| **Text Medium** | `#6B6B6B` | Texto secundário, labels, disclaimers |
| **Success** | `#7CB68E` | Checkmarks, progresso, confirmação |
| **CTA Primary** | `#C97B7B` | Botão principal de compra (rosa mais forte) |
| **CTA Hover** | `#B56A6A` | Hover state do botão |

### Regras de Contraste
- Texto sobre fundo claro: mínimo 4.5:1 (WCAG AA)
- CTA sobre fundo: mínimo 3:1
- Nunca texto claro sobre fundo claro

## 2. TIPOGRAFIA

| Uso | Fonte | Peso | Tamanho (mobile) |
|-----|-------|------|:----------------:|
| Headlines | **Playfair Display** | Bold (700) | 28-36px |
| Subheadlines | **Playfair Display** | Regular (400) | 20-24px |
| Body | **Inter** | Regular (400) | 16-18px |
| Body bold | **Inter** | SemiBold (600) | 16-18px |
| Labels/small | **Inter** | Medium (500) | 12-14px |
| CTA botão | **Inter** | Bold (700) | 18-20px |

**Regras:**
- Serif (Playfair) = títulos e frases emocionais
- Sans-serif (Inter) = corpo, instruções, CTAs
- Line-height: 1.5 para corpo, 1.2 para headlines
- Nunca ALL CAPS em body text (apenas headlines curtas)

## 3. ESTILO FOTOGRÁFICO

### Sujeito
- Mulheres latinas, 28-40 anos
- Corpo natural (não fitness model, não sobrepeso extremo)
- Pele morena a média (reflete LATAM)
- Cabelo escuro (preto ou castanho)
- Expressão: calorosa, genuína, confiante OU reflexiva (depende do contexto)

### Ambiente
- Doméstico: sala de estar, quarto, varanda
- Iluminação natural (janela, golden hour)
- Tons quentes — nunca iluminação fria/fluorescente
- Clean, organizado, mas real (não catálogo de móveis)

### Roupa
- Casual confortável: legging + top, vestido simples, camiseta
- Cores neutras/suaves (combina com paleta)
- NUNCA: biquíni, roupa fitness agressiva (neon/preto), roupas reveladoras

### Mood
- Acolhimento, não academia
- Calma, não urgência
- Empoderamento suave, não "girl boss"
- Referências: Noom, Calm app, Ritual vitamins

### PROIBIDO
- Antes/depois corporal explícito
- Barriga exposta em contexto negativo
- Imagens que pareçam de hospital/clínica
- Mulheres caucasianas/nórdicas (persona é latina)
- Fitness models com six-pack

## 4. ESTILO DE ILUSTRAÇÃO

### Para exercícios e anatomia (PDFs e site)
- Estilo: vetorial limpo, linha fina, minimalista
- Cores: da paleta (blush + nude + gold como acentos)
- Corpo: feminino, proporcional, sem detalhes excessivos
- Posições: mostrando o exercício de forma clara e instrucional
- Background: branco ou nude claro
- Referência: ilustrações de apps como Down Dog, Peloton stretching

### Para infográficos (e-book e TSL)
- Estilo: flat design, ícones simples
- Cores: paleta reduzida (máx 3 + neutros)
- Tipografia integrada
- Sem ornamentos desnecessários

## 5. LAYOUT — SPECS POR PEÇA

### Ads (Meta)

| Formato | Dimensão | Uso |
|---------|----------|-----|
| Feed quadrado | 1080x1080 | Imagem estática, carrossel |
| Story/Reel | 1080x1920 | Vídeo vertical, stories |
| Link ad | 1200x628 | Link para quiz |

**Regras de ad:**
- Texto na imagem: máx 20% da área (regra Meta)
- Hook visual no terço superior
- CTA no terço inferior
- Logo: pequeno, canto inferior direito (se usar)
- Fonte em imagens: Inter Bold, branco com sombra sutil ou caixa colorida

### PDFs

| Spec | Valor |
|------|-------|
| Tamanho | A4 (210x297mm) |
| Margem | 15mm todas as bordas |
| Grid | 2 colunas para texto, 1 coluna para imagens full |
| Fonte corpo | Inter 11pt mínimo |
| Fonte título | Playfair Display 18-24pt |
| Resolução imagens | 150dpi mínimo |
| Exportação | PDF otimizado para web (<5MB por arquivo) |

### Landing Page (TSL)

| Spec | Valor |
|------|-------|
| Width máxima | 680px (conteúdo) |
| Background | White Warm (#FFFAF5) |
| Seções alternadas | Nude Warm (#F5E6D3) |
| CTA botão | CTA Primary (#C97B7B), rounded 8px, padding 16px 32px |
| Espaçamento seções | 48-64px |
| Mobile padding | 16px lateral |
