# Offer Validation Checklist

Checklist master validado pelo @commander antes de aprovar a oferta completa para lançamento.

## Inteligência de Mercado
- [ ] offer-thesis.md completa e aprovada
- [ ] Mínimo 10 dores documentadas com citações do público
- [ ] Mínimo 10 desejos documentados com citações
- [ ] Análise de 3+ concorrentes realizada
- [ ] GAPs de mercado identificados
- [ ] Ângulos de copy definidos e priorizados
- [ ] Posicionamento diferenciado documentado

## Produto
- [ ] product-blueprint.md completo e aprovado
- [ ] Formato do produto definido e justificado
- [ ] Outline de conteúdo estruturado
- [ ] Bônus definidos (3-5 bônus complementares)
- [ ] Stack de valor > 5x preço do produto
- [ ] Quick win identificado (resultado rápido para o comprador)
- [ ] Plano de entrega definido

## Copy
- [ ] Hooks gerados (mínimo 15 variações)
- [ ] Sales letter escrita e aprovada pelo @copy-chief
- [ ] VSL script escrito (se funil usa vídeo)
- [ ] Email sequences definidas (boas-vindas, nurture, abandono)
- [ ] Quiz copy pronto (se quiz funnel)
- [ ] Todas as peças passaram no copy review checklist

## Criativos
- [ ] Identidade visual definida
- [ ] Creative briefs prontos (mínimo 5 criativos)
- [ ] Prompts de AI art gerados
- [ ] Variações por formato (feed, stories, reels)
- [ ] Criativos aprovados pelo @creative-director

## Funil
- [ ] Funil arquitetado e documentado
- [ ] Todas as páginas especificadas
- [ ] Order bump definido
- [ ] Upsell/downsell definidos
- [ ] Ticket médio estimado calculado
- [ ] Funil passou no funnel launch checklist

## Tráfego
- [ ] Unit economics positivo (lucro por venda > 0)
- [ ] CPA target definido e realista
- [ ] Plano de mídia com orçamento alocado
- [ ] Campanhas estruturadas (mínimo 3 adsets x 3 ads)
- [ ] Públicos definidos sem sobreposição
- [ ] Kill rules e scale rules documentadas
- [ ] Campanha passou no campaign launch checklist

## Infraestrutura
- [ ] Plataforma de pagamento configurada
- [ ] Email marketing integrado
- [ ] Tracking completo (pixels, events, UTMs)
- [ ] Automações testadas
- [ ] Área de membros/entrega configurada

---

## Veredicto Final

| Score | Decisão |
|:-----:|---------|
| 100% checks | 🚀 **LAUNCH READY** — Oferta pronta para lançamento |
| 90-99% | ⚠️ **CONDITIONAL** — Documentar gaps, lançar com monitoramento |
| 80-89% | 🟡 **NEEDS WORK** — Itens críticos pendentes |
| < 80% | 🔴 **NOT READY** — Retornar aos departamentos responsáveis |
