## Dev Notes

### Schema `intake_tokens` derivado dos routes

```sql
id                UUID PK
token             TEXT UNIQUE NOT NULL       -- 48 chars hex (crypto.randomBytes(24))
psychologist_id   UUID FK psychologists
patient_name_hint TEXT nullable              -- dica opcional de nome
expires_at        TIMESTAMPTZ NOT NULL       -- now + 7 days
used              BOOLEAN DEFAULT false
used_at           TIMESTAMPTZ nullable       -- set on submit
created_at        TIMESTAMPTZ DEFAULT now()
```

### Fix cron trial-expiry

Supabase retorna join de `psychologists(email, full_name)` como `Array | Object` dependendo
da cardinalidade da relação. O cast correto trata ambos os casos:

```ts
const psyRaw = sub.psychologists as { email: string; full_name: string }[] | { email: string; full_name: string } | null
const psy = Array.isArray(psyRaw) ? psyRaw[0] : psyRaw
```
