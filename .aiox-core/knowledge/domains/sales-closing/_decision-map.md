# Decision Map — Sales & Closing

## Quando Usar Qual Expert

### Para fechar por DM (sem call)
- Ticket abaixo de R$5k, audiencia organica → Expert: Doug | Framework: DM 8 mensagens + fechamento invisivel
- Ticket R$5k-R$10k → Expert: Doug | Framework: DM + Loom personalizado (video curto de diagnostico)
- Levantada de mao organica → Expert: Doug | Framework: ascensao low-to-high sem pressao

### Para fechar em call de vendas
- Ticket acima de R$10k → Expert: Miner + Gordon | Framework: NEPQ (Miner) + script de qualificacao (Gordon)
- Alta objecao / resistencia emocional → Expert: Miner | Framework: perguntas neuro-emocionais NEPQ
- Escalar com equipe setter/closer → Expert: Gordon | Framework: setter/closer model + script estruturado

### Para qualificar leads antes do fechamento
- Call agendada → Expert: Gordon | Framework: script de qualificacao pre-call
- DM inbound → Expert: Doug | Framework: 3 perguntas de diagnose antes de qualquer pitch

### Para lidar com objecoes
- Na call → Expert: Miner | Framework: reverter objecao com pergunta (nao com argumento)
- Na DM → Expert: Doug | Framework: nao rebater, aprofundar com pergunta de contexto
