# Decision Map — Copy & Persuasion

## Quando Usar Qual Expert

### Para escrever copy nova (organico — feed, stories, DM)
- Perfil sem narrativa definida → Expert: **Doug** | Framework: PMN → Big Idea → Primeira Linha 3 Fases
- Narrativa definida, precisa de tensao → Expert: **Doug** | Framework: SHIFT_TENSION / RESET_TENSION
- Audiencia fria, contexto desconhecido → Expert: **Blair Warren** | Framework: 5 Drivers
- Escrevendo bio/posicionamento → Expert: **Doug** | Framework: Inimigo Comum + Big Idea

### Para escrever copy para trafego pago (ads)
- Volume de criativos urgente (10+ variacoes) → Expert: **Kobata** | Framework: Ed 100x + Lateralizacao
- VSL / video de vendas → Expert: **Hard Copy** | Framework: Kinshu 4-Act
- Copy de email / long-form → Expert: **Agora Inc** | Framework: 4Ps + Big Idea
- Pagina de vendas estruturada → Expert: **Jim Edwards** | Framework: PAS/PASTOR/QUESTS

### Para validar ou auditar copy existente
- Copy que nao converte → Expert: **Doug** | Framework: Prompt Proibido + revisar PMN
- Auditoria de promessa/posicionamento → Expert: **Agora Inc** | Framework: Reason-Why Copy
- Teste de hook → Expert: **Kobata** | Framework: Hook Rate Benchmark (55%+)
- Auditoria completa → Expert: **Doug** | Framework: PromptBook Cat 4 (Auditoria)

## Tensoes Conhecidas Neste Dominio

| Situacao | Expert A | Expert B | Criterio de Desempate |
|----------|---------|---------|----------------------|
| Volume vs. qualidade de copy | Kobata: 50+ criativos/semana, variar ganchos | Doug: proporcao T.A.D. (Tensao 50%, qualidade intencional) | Narrativa nova → Doug. Oferta validada para escalar → Kobata |
| Estrutura rigida vs. principios fluidos | Jim Edwards: PAS/PASTOR fixo | Kobata: principios > scripts | Copy de resposta direta (ads) → Edwards. Conteudo organico → Kobata |
| Hook como gancho emocional vs. como filtro | Doug: Inversao de Procura (declaracao de dominio) | Hard Copy: hook como filtro de 3s (quem deve assistir) | Instagram/organico → Doug. Video/VSL pago → Hard Copy |
| Copy tecnica vs. copy emocional | Agora Inc: especificidade e credibilidade ($387.42) | Blair Warren: emocao e validacao antes de logica | Anuncio com metrica → Agora. Venda relacional → Warren |
