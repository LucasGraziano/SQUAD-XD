# Decision Map — Offers & Pricing

## Quando Usar Qual Expert

### Para criar uma nova oferta do zero
- Produto novo, sem audiencia previa → Expert: Hormozi | Framework: Grand Slam Offer + Value Equation
- Produto novo com funil de ads → Expert: Brunson | Framework: Hook-Story-Offer + bump/upsell sequencial
- Produto novo organico com audiencia pequena → Expert: Doug | Framework: micro-produto como porta de entrada

### Para precificar uma oferta existente
- Produto com alta demanda, lista de espera → Expert: Haynes | Framework: pricing evolution (dobrar preco a cada 60 dias)
- Produto estabelecido com multiplos niveis → Expert: Doug | Framework: Teia de Ofertas + Niveis de Acesso
- Repositionamento de commodity → Expert: Hormozi | Framework: Value Equation (aumentar valor percebido)

### Para escalar receita sem novo produto
- Audiencia estabelecida → Expert: Doug | Framework: Doc do Dominio + Loop de Compras
- Base de clientes com historico de compra → Expert: Brunson | Framework: Funnel Stacking + continuidade
- Demanda organica crescente → Expert: Haynes | Framework: demand signals + pricing evolution

### Para criar nivel de acesso premium
- Velocidade e exclusividade como diferencial → Expert: Doug | Framework: Niveis de Acesso
- High-ticket com entrega intensa → Expert: Hormozi | Framework: stacking de bonus + garantia inversa
