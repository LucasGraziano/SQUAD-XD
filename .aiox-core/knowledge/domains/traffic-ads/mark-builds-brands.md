## L1 Filosofias
- Identidade cria realidade — resultados externos refletem o sistema de crenças interno; consertar identidade antes de aplicar táticas; broke identity é instalada inconscientemente desde o nascimento
- Contraste é a fundação da diferenciação — quando o mercado inteiro faz X, faça Y; posicionamento monopolista (Zero to One) aplicado ao criativo; mad scientist thinking frame
- Copy é a fundação de tudo no e-commerce — criativos, funis, ads, landing pages: todos downstream de copy; um winning ad muda tudo e seu fundamento é copy
- AI é amplificador, não substituto — AI + habilidade existente = velocidade e escala exponenciais; obsessão com AI é não-negociável; quem negligencia AI não quer riqueza o suficiente
- Marketing é, em sua essência, mudança de crença — o trabalho é pegar as crenças existentes do cliente e alinhar com as crenças que geram compra; nada mais, nada menos
- Research first, sempre — gatilhos emocionais devem ser validados antes de qualquer ad spend; adivinhar é obsoleto quando existem ferramentas de AI para validar
- Clientes > concorrentes — voz direta do cliente é dado mais puro do que espionar concorrente; concorrentes são copycats copiando copycats (hasty generalization + survivorship bias)
- Velocidade compõe vantagem — money loves speed; iteração rápida de criativos compõe vantagem competitiva; consumir menos e criar mais é o KPI de impacto pessoal

## L2 Modelos Mentais
- **Eugene Schwartz Market Sophistication (estágios 1-5)**: calibrar copy e contraste criativo de acordo com a sofisticação do mercado — mercado avançado (estágio 4-5) exige mecanismo único e contraste de formato, não apenas promessa
- **Transformation Hook Duality**: hook de imagem existe em dois formatos — transformação completa (pesadelo→sonho) ou pesadelo-only; a emoção vive no contraste entre estados; transformação completa tem maior poder emocional mas pesadelo-only é mais versátil
- **Emotionally Primed State**: quiz como veículo para colocar comprador em estado hipnótico de "sim" antes de qualquer oferta ser apresentada; estado de alta sugestibilidade = crédito na mão, pronto para comprar
- **Identity Reprogramming Frame**: crenças de dinheiro são instaladas inconscientemente; reprogramação via declarações "I'm the type of person that..." (presente, específico) + say/do ratio para construir auto-confiança por micro-ações
- **Creator vs. Consumer Ratio**: medir impacto pessoal pela proporção criação/consumo nas últimas 24h; líderes ao longo da história criam — não apenas consomem
- **Survivorship + Hasty Generalization**: "funciona para eles então funciona para mim" é falácia lógica; diferentes variáveis de negócio, diferentes resultados; validar com dados próprios antes de copiar

## L3 Heurísticas
- 80/20 de teste criativo: 80% iterações de formatos existentes vencedores, 20% conceitos completamente novos
- Threshold de fadiga de imagem: não preocupar com fadiga criativa até $10K-$50K de gasto por criativo específico
- Regra de variação de formato no quiz: nunca mais de 2-3 perguntas consecutivas no mesmo formato para manter dopamina e engajamento
- 4 documentos fundacionais obrigatórios: research doc + avatar doc + offer doc + beliefs doc antes de qualquer produção criativa
- 3 princípios de quiz obrigatórios: micro-commitments + seeding + pre-handling objections — os 3 devem estar presentes em todo quiz que escala
- Gauge de sofisticação de mercado: quando >50% do mercado faz X, posição de contraste tem maior probabilidade de breakthrough
- Gradualization em quiz: perguntas começam não-invasivas (objetivo/goal) e escalam para invasivas (idade, peso); nunca inverter sequência
- Batch de criativos AI: 20 criativos de alta qualidade por batch em menos de 1 hora com workflow correto de AI

## L4 Frameworks
- **Foundational Docs System** (5 etapas): Research Doc (AI deep research em voz do cliente — quotes verbatim, linguagem de transformação, narrativa de dor) → Avatar Doc (demografia, psicografia, desejos, medos, crenças atuais) → Offer Doc (mecanismo, proposta única, posicionamento de contraste) → Beliefs Doc (crenças atuais → crenças-alvo → crenças ponte a seedar) → Armazenar como 4 DOCX para alimentar todo criativo, funil e copy projeto
  - `USE:` antes de qualquer produção criativa, construção de funil ou projeto de copy — foundational docs são o input que determina qualidade do output
  - `FAIL:` pular research → adivinhar gatilhos emocionais → ads genéricos sem trigger específico → desperdiçar budget em criativos que não convertem
  - `VS:` outros criadores começam pela creative; Mark começa pelos dados do cliente — research-first é o diferencial que permite criar winning creatives sistematicamente
- **AI Image Ad Process** (6 etapas): Foundational docs (research → avatar → offer → beliefs) → Identificar formatos provados via Get Hooked (4-5 estrelas, still running, mesmo nicho) → Selecionar ângulo de hook (transformação completa ou pesadelo-only) → Usar AI para escrever o prompt de geração de imagem (não só para executar) → Batch criar 20 criativos → 80/20 split (80% variações de formatos provados, 20% conceitos novos)
  - `USE:` marcas de e-commerce que querem teste criativo rápido e de alto volume sem designer
  - `FAIL:` copiar ad da biblioteca → pedir ao AI para refazer "para minha marca" → output genérico sem trigger emocional → jogar no ad account sem validação → questionar por que não vira milionário
  - `VS:` Haynes usa Tornado Testing (50+ criativos com kill em 24-48h); Mark usa batch de 20 com validação de trigger antes; complementares — Mark para validade emocional, Haynes para volume e velocidade
- **Quiz Funnel Build** (6 etapas): Foundational docs (research, avatar, offer, beliefs) → Mapear crenças do cliente (identificar quais seedar) → Mapear objeções (identificar quais pre-handle via intra-quiz cards) → Escrever perguntas usando gradualization (não-invasivo primeiro, invasivo depois) → Design dopaminogênico de formatos variados (multiple choice, sliders, yes/no — nunca 2+ iguais seguidos) → Adicionar intra-quiz cards entre perguntas para seedar crenças e pre-handle objeções
  - `USE:` quando mercado sofisticado e envio direto para product page não converte; esconder o processo de venda por trás do "Take the quiz" CTA tem invasividade dramatically menor que "Shop now"
  - `FAIL:` perguntas aleatórias sem belief-mapping → quiz falha em primar emocionalmente o comprador → taxa de conversão não melhora sobre envio direto
  - `VS:` Russell Brunson foca em value ladder e bump/upsell pós-quiz; Mark foca na arquitetura psicológica do quiz em si (emotional priming como mecanismo central)
- **Identity Reprogramming** (3 etapas): Auditar crenças inconscientes atuais de dinheiro/sucesso (surfaçar o que foi instalado) → Escrever declarações de identidade nova — "I'm the type of person that..." (presente, específico, repetido) → Executar say/do ratio — só fazer compromissos que serão cumpridos; construir auto-confiança por micro-ações antes de tentar grandes mudanças
  - `USE:` antes de qualquer modelo de negócio, tática ou mudança de estratégia; identidade é fundação de todos os resultados
  - `FAIL:` aplicar táticas sem audit de identidade → resultados externos continuam refletindo crenças internas quebradas → ciclo de frustração tática
- **Contrast Marketing Strategy** (4 etapas): Mapear sophistication stage do nicho (Schwartz 1-5) → Catalogar o que 80% dos concorrentes fazem (formato, tipo de hook, ângulo de oferta, estilo criativo) → Identificar posição oposta ou adjacente (Zero to One + mad scientist thinking) → Construir criativo baseado no contraste — mecanismo único, formato inesperado, ângulo polarizador
  - `USE:` entrando em mercados saturados ou quando ângulo criativo atual fatigou; contraste é o moat de diferenciação no estágio 4-5 de sofisticação
  - `FAIL:` contraste por contraste sem ressonância emocional → estranho mas não converte; contraste precisa ser combinado com trigger emocional do foundational docs

## L5 Metodologias
- **Branded Dropshipping at Scale**: Sistema completo de seleção de nicho → sourcing de produto → construção de marca → produção criativa AI-native → arquitetura de quiz funnel → media buying → CRO → scaling; copy como fundação em cada camada; AI aplicada a customer research, produção criativa, landing pages e operações de backend; validado em ~$10M de receita gerada
- **AI-Native Creative Production System**: Foundational docs (4 documentos via AI deep research) → market research via AI (validação de trigger antes de spend) → batch de produção de criativos (20 image ads/hora com workflow correto) → split 80/20 teste (iterações de winners + novos conceitos) → biblioteca de formatos provados × ângulos novos → kill/scale baseado em thresholds de spend ($10K-$50K por criativo antes de preocupar com fadiga)
- **Quiz Funnel Revenue System**: Belief mapping de cliente → arquitetura de quiz (micro-commitments + seeding + pre-handling) → design de estado emocionalmente primado via variações de formato dopaminogênicas → reveal de oferta com fricção mínima (CTA "Take the quiz" vs. "Shop now") → scaling via tráfego pago para o quiz; validado em $1M/30 dias; contexto: janela de oportunidade limitada para arquiteturas de funil novas — urgência é estratégica

## Anti-Patterns
- **Copy-first sem Research**: criar criativos baseados em intuição ou ad library → output genérico sem trigger emocional específico → CPM desperdiçado → **Fix:** foundational docs primeiro, sempre; AI deep research valida qual transformação ressoa antes de qualquer spend
- **Copycat de Concorrente**: assumir que o que funciona para concorrente funciona para você (hasty generalization) → produtos e mercados diferentes, resultados diferentes → **Fix:** falar com seus próprios clientes; "o que quase te impediu de comprar?" é a pergunta mais valiosa
- **Quiz sem os 3 Princípios**: quiz como formulário de perguntas sem micro-commitments deliberados, seeding de crenças e pre-handling de objeções → não gera estado emocionalmente primado → conversion rate igual ou pior que direct-to-page → **Fix:** mapear crenças e objeções antes de escrever qualquer pergunta
- **AI sem Processo**: usar AI para gerar copy/criativos sem foundational docs → output genérico de "AI slop" sem trigger emocional → **Fix:** AI é amplificador — o input (foundational docs + pesquisa) determina a qualidade do output; garbage in, garbage out
- **Tática antes de Identidade**: aplicar novo modelo de negócio ou estratégia sem auditar identidade de dinheiro → resultados externos continuam refletindo crenças internas → ciclo de "tentar e fracassar" perpetua → **Fix:** identity audit é o pre-requisito #1 antes de qualquer mudança de tática
