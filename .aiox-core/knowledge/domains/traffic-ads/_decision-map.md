# Decision Map — Traffic & Ads

## Quando Usar Qual Expert

### Para comecar do zero (sem audiencia)
- Trafego pago imediato → Expert: Haynes | Framework: DSL + creative testing + Hammer Strategy
- Escala rapida Facebook/Meta → Expert: Hard Copy | Framework: Kinshu VSL + kill rules + escala R$50 a R$500
- Construcao de audiencia organica → Expert: Doug | Framework: StoryAds + Money Brand + Z4

### Para escalar o que ja funciona
- Criativo validado, aumentar budget → Expert: Hard Copy + Haynes | Framework: escala gradual com kill rules
- Ops e estrutura para escalar → Expert: Linder | Framework: Scorecard hiring + sistemas de contratacao
- Automacao de ops com IA → Expert: Jordan Lee | Framework: estrutura de custo variavel + AI-driven ops

### Para resolver audiencia desqualificada
- Muitos seguidores, poucos compradores → Expert: Doug | Framework: Money Brand + Z4 (narrativa de qualificacao)
- Leads que nao aparecem → Expert: Haynes | Framework: Hammer Strategy (12 emails + 4 SMS em 48h)

### Para testar criativos
- Qual variavel testar primeiro → Expert: Haynes | Framework: criativo e variavel #1 (antes de audiencia)
- Formato de criativo para ads frios → Expert: Hard Copy | Framework: Kinshu VSL structure
